# marcosantonio-457000.github.io

Este portafolio fue elaborado con el propósito de mostrar de manera 
organizada, visual y profesional las competencias, proyectos y experiencias 
que he adquirido a lo largo de mi formación académica y práctica en el área de 
Ingeniería en Sistemas. Su diseño fue desarrollado utilizando tecnologías web 
como HTML y CSS , priorizando una interfaz limpia, moderna y 
completamente adaptable a diferentes dispositivos. 
La estructura del sitio se construyó con una navegación clara que permite al 
usuario explorar mis proyectos más relevantes, revisar mi experiencia 
profesional y acceder directamente a mi currículum en formato digital. Se 
integraron también elementos visuales, iconografía y enlaces interactivos para 
mejorar la experiencia de uso y lograr una presentación más dinámica e 
intuitiva. 
Este portafolio tiene como finalidad servir como una carta de presentación 
digital ante posibles empleadores, colaboradores o instituciones académicas. 
Fue diseñado para reflejar mi identidad profesional, evidenciar mis 
capacidades técnicas y demostrar mi interés por el aprendizaje continuo y la 
innovación tecnológica. Asimismo, representa un espacio en constante 
evolución, donde seguiré incorporando nuevos conocimientos, habilidades y 
proyectos que fortalezcan mi crecimiento dentro del campo de la tecnología.
